# AutoOp

A small Spigot/Paper plugin that grants OP to the server owner on join.

## Requirements
- Java 17+
- Maven
- Spigot/Paper 1.20+

## Build
    mvn clean package

The compiled jar will be in `target/autoop-1.0.jar`.

## Install
1. Drop the jar into your server's `plugins/` folder.
2. Restart the server.

## Note
For use by the server owner on their own server.
